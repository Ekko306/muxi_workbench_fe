Avatar——头像组件

【props】

|  名称  |            描述            | type    |
|:------:|:--------------------------:|---------|
|  width |          头像宽度 (单位："px"; 例如："20" ; 默认："44")         | string  |
| height |          头像高度 (同上)         | string  |
|   src  |    图片地址（同img标签）   | string  |
| square | 是否是正方形（默认为圆形） | boolean |
