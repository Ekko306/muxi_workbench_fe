Button——按钮组件

【props】

|  名称  |            描述            | type    |
|:------:|:--------------------------:|---------|
|  width |          button宽度  （如："40"， 默认为96px）        | string  |
| height |          button高度  （同上，默认为40px）        | string  |
|   text  |    显示文字            | string  |
| square | 是否是正方形（默认为圆形） | boolean |
| to | 同Link标签的"to"属性 | string |
