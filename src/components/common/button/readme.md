Button——按钮组件

【props】

|  名称  |            描述            | type    |
|:------:|:--------------------------:|---------|
|  width |          button宽度  （默认为96px）        | string  |
| height |          button高度  （默认为40px）        | string  |
|   text  |    显示文字            | string  |
| square | 是否是正方形（默认为圆形） | boolean |
